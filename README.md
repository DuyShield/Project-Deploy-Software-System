# Flower Cat Shop - Website Quản lý & Bán hoa trực tuyến

## Giới thiệu
Flower Cat Shop là dự án website thương mại điện tử được xây dựng theo mô hình **PHP MVC** thuần. Dự án tập trung vào việc cung cấp trải nghiệm mua sắm mượt mà cho người dùng và hệ thống quản lý kho hàng mạnh mẽ cho người quản trị.

## Tính năng nổi bật
* **Phía Người dùng:**
    * Quản lý thông tin tài khoản. 
    * Duyệt sản phẩm theo danh mục,lọc sản phẩm theo loại, tìm kiếm và xem chi tiết hoa.
    * Giỏ hàng (Cart) và quy trình thanh toán (Checkout).
    * Tài khoản người dùng với chức năng đăng nhập "Remember Me" qua Cookie/Session.
    * Tích hợp **Provinces API** để tự động chọn Tỉnh/Thành phố khi đặt hàng.
    * Lưu thông tin địa chỉ nhận hàng.
    * Quản lý đơn đã đặt hàng.
* **Phía Quản trị (Admin Dashboard):**
    * Quản lý sản phẩm (CRUD): Thêm mới, cập nhật hình ảnh và số lượng hoa.
    * Quản lý đơn hàng: Theo dõi trạng thái đơn hàng của khách hàng.
    * Quản lý danh sách tin nhắn liên hệ của khách hàng.
    * Quản lý danh sách đơn đặt hàng.
    * Quản lý kho hàng sản phẩm tồn kho.
    * Thống kê báo cáo cơ bản.
## 🛠️ Công nghệ sử dụng
* **Ngôn ngữ:** PHP (mô hình MVC).
* **Giao diện:** HTML5, CSS3, Bootstrap 5, JavaScript.
* **Cơ sở dữ liệu:** MySQL (Quản lý qua Laragon/HeidiSQL).
* **Khác:** JSON, AJAX (để load địa chỉ), Git.
